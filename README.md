# Hotel Booking Management System

A simple web-based hotel room booking system built using:
- PHP
- MySQL
- HTML, CSS, Bootstrap
- XAMPP for local server

## Features
- User registration and login
- Room booking
- User dashboard
- Admin panel to manage bookings

## How to Run
1. Install XAMPP.
2. Copy the project to `htdocs`.
3. Import the database SQL file into phpMyAdmin.
4. Start Apache & MySQL.
5. Open: http://localhost/hotel-booking/
